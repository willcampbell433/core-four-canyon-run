# Core Four Canyon Run

A public-safe commemorative trip dashboard for the July 3-4 offshore tuna run from Brick, NJ.

## What it includes

- Trip command screen
- Approximate route map with no real numbers
- Cabin TV mode
- Crew cards
- Quick local trip log
- Bird-watch controls
- Recap and prep board

This is intentionally static so it can be hosted on GitHub Pages.
